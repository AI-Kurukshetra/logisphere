🚀 QUICK START (5 MINUTES)

1. INSTALL DEPENDENCIES
   $ npm install puppeteer remotion
   
2. START DEV SERVER (in another terminal)
   $ npm run dev
   
3. RECORD SCREENSHOTS
   $ npx ts-node scripts/video-walkthrough/puppeteer-recorder.ts
   Output: 100+ PNG files + metadata.json (~2-5 minutes)

   $ npx remotion render \

🎨 CUSTOMIZATION EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

CHANGE NARRATION:
  → Edit scripts in page-scripts.ts
  → Re-run puppeteer-recorder.ts and render

ADD AUDIO:
  → Run tts-generator.ts before rendering
  → Choose provider: Google (free), ElevenLabs (pro), AWS (pro)

ADJUST VIDEO QUALITY:
  → Use CRF flags: CRF 16 (max) to CRF 30 (low)
  → Or switch codec: H.264 → VP9 → ProRes

ADD WATERMARK/BRANDING:
  → Edit remotion-composer.tsx OpeningSequence
  → Add logo image, custom colors, fonts

EXTEND TO MORE PAGES:
  → Add new entry to pageScripts array
  → Define narration and scrolls
  → Re-run recorder and renderer
